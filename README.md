# gojsonstruct

Converts json documents into struct 

##Example usage 


{
    "person" : {
        "name" : "John",
        "age" : 30,
        "cars" : {
            "car1" : "BMW",
            "car2" : "Tesla"
        },
        "favouriteLanguages": ["Go","Java","JS"]
    },
    "computer" : {
        "name" : "Apple",
        "year" : 2019,
        "new" : true,
        "version": 10.1
    }
}
